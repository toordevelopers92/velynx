# Contributing

Thank you for helping improve Velynx.

## Development

```bash
pnpm install
pnpm build
pnpm test
pnpm docs:build
```

## Guidelines

- Keep changes focused and documented.
- Prefer additive utilities over breaking changes.
- Ensure components remain keyboard accessible.
- Update docs and examples when behavior changes.

## Pull requests

- Describe the goal and scope.
- Include before/after notes or screenshots if UI changes.
- Add tests for interaction modules or generator behavior.
